ℚℝ 3 Visualização de Dados

Muitas vezes, os detalhes da informação que estamos tentando compreender, que podem estar ocultos nos dados, não são facilmente capturados apenas por análises e processos numéricos.

Visualizar dados em gráficos é crucial por várias razões:

  • Facilita a Interpretação: Gráficos tornam os dados mais compreensíveis ao transformar números em tabelas em representações visuais que, em geral, são mais fáceis de interpretar. Pois, isso ajuda a identificar padrões (tendências), anomalias (outliers) e relação entre variáveis de forma mais intuitiva. É mais fácil entender e analisar informações visuais do que interpretar grandes conjuntos de números.

  • Facilita a Análise Exploratória: Durante a análise exploratória dos dados, gráficos ajudam a entender melhor o conjunto de dados e a formular hipóteses para testes estatísticos subsequentes.

  • Ajuda na Comparação: Permite a comparação direta entre diferentes conjuntos de dados ou categorias.

  • Comunicação: Facilita a comunicação quando você precisar apresentar seus resultados para outras pessoas.

Em termos gerais, os gráficos são úteis para tornar os dados mais compreensíveis, detectar padrões e tendências.

Quarteto de Anscomble

O Quarteto de Anscomble20 é um conjunto de dados artificial criado justamente com a intenção de, forma gráfica, transmitir ao menos algumas das justificativas anteriores para obter uma compreensão completa da informação neles contida.

Esse conjunto de dados foi projetado de maneira a ter algumas estatísticas descritivas númericas idênticas (média, variância, correlação, reta) mais com diferentes padrões óbvios quando visualizados graficamente. Fica evidente que dados podem ter estatísticas numéricas similares mas podem ter estruturas muito diversas. A tabela a seguir mostram esses dados.

Construindo bons Gráficos…

Um bom gráfico deve comunicar informações de maneira clara e direta. Para alcançar esse objetivo, é fundamental que ele possua certas características essenciais. A seguir, apresentamos as principais características que contribuem para a eficácia de um gráfico.

  • Clareza: As informações devem ser apresentadas de maneira que sejam facilmente compreendidas pelo público. Evite a sobrecarga de elementos no gráfico, como excessos de dados, linhas ou legendas, pois isso pode confundir o espectador. Um design limpo e focado ajuda a transmitir a mensagem de forma mais eficaz.

  • Título Descritivo: Um título deve ser claro e conciso, refletindo precisamente o conteúdo e o propósito do gráfico. Ele deve fornecer uma visão imediata sobre o que está sendo apresentado, ajudando o espectador a entender rapidamente o tema abordado e o contexto dos dados.

  • Eixos Bem Definidos: Os eixos do gráfico devem ter rótulos claros e informativos, que identifiquem claramente o que cada um representa. É essencial incluir as unidades de medida quando aplicável, pois isso ajuda o espectador a interpretar os dados com precisão e a entender a escala e o contexto das informações apresentadas.

  • Legibilidade: Escolha fontes que sejam claras e legíveis, utilizando tamanhos apropriados para garantir que todos os elementos do gráfico sejam facilmente compreensíveis. A legibilidade é fundamental para que o público possa absorver as informações rapidamente, sem esforço adicional para decifrar texto ou números. Esteja ciente de que um gráfico bonito é tão ruim quanto um gráfico feio pois elementos de estética não devem nunca se sobreporem aos elementos informativos! Então evite qualquer fonte decorativa que possa prejudicar a clareza.

  • Cores e Estilos: Utilize cores que contrastem bem entre si para facilitar a distinção entre diferentes elementos do gráfico. Quando o gráfico precisar ser impresso em cores e em escala de cinza, escolha uma paleta que mantenha a clareza das informações em ambas as versões. Isso garante que os dados sejam legíveis e facilmente interpretáveis, independentemente do formato de impressão. Existem paletas de cores projetadas especificamente para minimizar a perda de informação ao converter o gráfico do espaço de cores para a escala de cinza. Essas paletas são cuidadosamente elaboradas para garantir que as diferenças e nuances importantes entre os dados sejam preservadas, mesmo quando visualizadas em preto e branco. Dessa forma, a legibilidade e a clareza do gráfico são mantidas, independentemente do formato de apresentação (Pesquise sobre a paleta de cores Viridis21 para saber mais).

    😈

  • Escala Apropriada: A escala do gráfico deve ser adequada ao tipo de dados apresentados, permitindo uma interpretação precisa e facilitando a comparação entre diferentes elementos. Escolher uma escala que represente adequadamente a variação dos dados ajuda a evitar distorções e a garantir que as diferenças sejam facilmente visíveis. Isso contribui para uma análise mais eficaz e intuitiva das informações.

  • Dados Precisos: Garanta que os dados representados no gráfico sejam corretos, confiáveis e atualizados. A precisão das informações é fundamental para a credibilidade do gráfico e para a tomada de decisões informadas. Verificar as fontes e a integridade dos dados antes de apresentá-los ajuda a evitar mal-entendidos e a transmitir uma mensagem clara e consistente.

  • Anotações e Destaques: Utilize anotações para realçar pontos-chave e informações relevantes no gráfico, facilitando a compreensão do espectador. No entanto, é importante evitar excessos; muitas anotações podem criar confusão e desviar a atenção do principal. O equilíbrio é essencial: destaque o que é realmente significativo para que a mensagem central permaneça clara e impactante.

  • Interatividade (se aplicável): Em gráficos digitais, a inclusão de elementos interativos pode enriquecer a experiência do usuário, permitindo uma exploração mais profunda dos dados. Funcionalidades como zoom, filtragem e visualizações dinâmicas permitem que o público interaja com as informações, descubra padrões e personalize a visualização de acordo com suas necessidades. Isso não só torna a análise mais envolvente, mas também ajuda na compreensão de insights complexos.

Essas características são essenciais para garantir que o público compreenda rapidamente a mensagem que o gráfico busca transmitir. Ao priorizar clareza, precisão e design eficaz, facilitamos a interpretação dos dados e potencializamos o impacto da informação, tornando-a mais acessível e significativa para todos.

3.1 Gráficos com o R

R oferece um rico ecossistema de funções e pacotes para a criação de vários tipos de gráficos que permite desde visualizações simples até representações sofisticadas para necessidades específicas. Desde o ínicio, nas próprias palavras do R Core Team, a definição para o R foi22:

“R é uma linguagem e ambiente para computação estatística e gráficos”.

E o cuidado com o sistema gráfico foi planejado desde a origem do ambiente:

“Um dos pontos fortes do R é a facilidade com que gráficos bem projetados com qualidade de publicação podem ser produzidos, incluindo símbolos matemáticos e fórmulas quando necessário. Foi tomado muito cuidado com os padrões para as pequenas opções de design em gráficos, mas o usuário mantém o controle total”.

A adoção e desenvenvolvimento posterior foi consequência direta dessa preocupação em se ter um software livre com tais capacidades também em quesitos de produção de boas imagens de gráficos estatísticos.

O site The R Graph Gallery23 possui uma coleção extensa de gráficos gerados por R na web. São centenas de gráficos são exibidos em diversas seções, sempre com seu código reproduzível disponível. H

Há também R Gallery Book24, um guia completo baseado no site da R Graph Gallery. Ele extrai informações diretamente da galeria, oferecendo detalhes minuciosos na reprodução e na elaboração de ideias.

3.1.2 Pacote animation

Com o uso do pacote animation é possível ir além dos gráficos estáticos, permitindo criar animações, proporcionando um modo dinâmico de visualizar os dados quando pertinente. É particularmente útil quando você deseja visualizar mudanças ou interações complexas ao longo do tempo ou ângulos, como simulações, evoluções temporais ou rotações de gráficos 3D.

O principal propósito do pacote animation é facilitar a criação de GIFs animados, vídeos e apresentações interativas diretamente a partir de gráficos gerados no R. Ele simplifica a criação de animações de maneira iterativa e automática, combinando vários gráficos em uma sequência que pode ser visualizada como um filme.

Funcionalidades e Vantagens:

  1. Criação de GIFs e Vídeos:
    • O animation permite salvar animações em diferentes formatos, como GIF, vídeo (MP4, AVI) e até slideshows em HTML.
    • É multi-formato: Suporta vários formatos de saída, como GIF, MP4, HTML, e AVI.
    • Funções como saveGIF(), saveVideo(), e saveHTML() facilitam o processo de salvar as animações em diversos formatos de mídia.
  2. Simulações Dinâmicas:
    • É ideal para criar animações de simulações estatísticas, como métodos de Monte Carlo, simulações de cadeia de Markov, evolução de dados ao longo do tempo ou visualizações rotativas de gráficos 3D.
  3. Controle Total da Animação:
    • O usuário pode controlar a velocidade, número de quadros e até personalizar os intervalos de tempo entre os quadros, proporcionando flexibilidade para criar animações simples ou avançadas.
    • Exemplos: ani.options(interval = 0.1) define o intervalo de tempo entre os quadros da animação.
  4. Facilidade de Uso:
    • A estrutura é intuitiva, mesmo para iniciantes. Para criar uma animação, basta colocar o código de criação dos gráficos dentro de um loop, e a função saveGIF() ou outra função correspondente cuidará do resto.
    • É simples e rápido pois mesmo com pouco código é possível criar animações sofisticadas.
  5. Integração com Gráficos 3D e Superfícies:
    • Funciona muito bem com gráficos tridimensionais, permitindo criar animações que rotacionam gráficos em torno de seus eixos ou mostram como eles evoluem em várias dimensões.
  6. Interatividade e Visualização de Modelos:
    • A animação pode ajudar a entender visualmente conceitos complexos, como a convergência de estimativas em modelos estatísticos, a simulação de distribuições probabilísticas, ou a visualização de resíduos em modelos de regressão.

3.1.3 Pacote ggplot2

O ggplot236, 37 é um dos pacotes mais renomados para visualização de dados em R, desenvolvido e mantido por Hadley Wickham. Baseado na Gramática dos Gráficos (Grammar of Graphics38), ele oferece uma abordagem estruturada e intuitiva para criar representações visuais sofisticadas e informativas. Ao combinar diferentes camadas, como pontos, linhas e barras, o ggplot2 proporciona grande flexibilidade na personalização dos gráficos, permitindo criar visualizações esteticamente agradáveis e adaptadas às suas necessidades.

O pacote ggplot2, por ter sua estrutura baseada em camadas, é altamente extensível. Essas extensões são desenvolvidas por uma comunidade ativa de usuários e desenvolvedores e são distribuídas como pacotes adicionais no ecossistema R. Elas podem adicionar novas funcionalidades, melhorar a estética dos gráficos ou facilitar o uso de ggplot2 em contextos específicos.

O site: https://exts.ggplot2.tidyverse.org/ rastreia e lista extensões do pacote ggplot2 desenvolvidas por usuários R na comunidade. O objetivo do site é facilitar aos usuários do R a localização dessas extensões desenvolvidas. A galeria39 de demonstração possui mais de 130 extensões, todas acompanhadas de exemplos para ajudar na sua usabilidade.

Gramática dos Gráficos

A Gramática dos Gráficos é um framework teórico para a construção de gráficos por meio de camadas.

Aqui está uma visão geral dessas camadas:

Conceitos Fundamentais

  1. Componentes Básicos:
    • Dados: O conjunto de dados a ser visualizado.
    • Estéticas (Aesthetics): Mapeamento de variáveis para propriedades visuais, como a posição, cor, tamanho e forma dos elementos do gráfico.
    • Geometria (Geoms): Define os tipos de gráficos e formas visuais representadas que você quer criar, como pontos (geom_point()), linhas (geom_line()), barras (geom_bar()), etc.
    • Facetas (Facets): Divisão dos dados em subconjuntos para criar múltiplos gráficos, dividindo os dados com base em uma ou mais variáveis para facilitar a comparação entre grupos (usando facet_wrap() ou facet_grid()).
    • Estatísticas (Stats): Transformações e cálculos realizados sobre os dados antes de serem visualizados, como a adição de linhas de tendência (usando stat_smooth()).
    • Coordenação (Coords): Sistema de coordenadas usado para posicionar os elementos no gráfico, como coordenadas cartesianas (coord_cartesian()) ou polares (coord_polar()).
    • Temas (Themes): Personalização da aparência estética do gráfico, incluindo a formatação de texto, linhas e elementos de fundo.
  2. Camadas (Layers):
    • Em vez de criar um gráfico de uma vez, a Gramática dos Gráficos permite construir gráficos adicionando camadas. Cada camada representa uma parte do gráfico e pode ser ajustada e combinada de maneira modular.
  3. Gráficos Combináveis:
    • Graças à abordagem modular, você pode combinar diferentes tipos de gráficos e camadas em uma única visualização. Por exemplo, adicionar uma linha de regressão a um gráfico de dispersão ou combinar gráficos de barras com gráficos de linha.
  4. Sistema de Mapeamento:
    • A Gramática dos Gráficos usa mapeamentos estéticos para associar variáveis dos dados a propriedades visuais dos elementos do gráfico. Isso facilita a criação de visualizações que são tanto informativas quanto visualmente impactantes.

Benefícios da Gramática dos Gráficos

  • Flexibilidade: Permite criar visualizações complexas combinando diferentes elementos.
  • Modularidade: Facilita a construção e modificação de gráficos por meio da adição ou remoção de camadas.
  • Consistência: Oferece uma abordagem consistente para a criação de gráficos, tornando mais fácil entender e interpretar a visualização.

O objetivo da Gramática dos Gráficos é facilitar a criação de representações visuais de maneira intuitiva e estruturada, promovendo uma análise mais direta e uma comunicação mais eficaz dos resultados.

3.1.4 Pacote plotly

O pacote plotly cria gráficos interativos e dinâmicos. Ele é baseado na biblioteca JavaScript Plotly.js, que é amplamente utilizada para visualizações de dados na WEB. O objetivo do plotly é facilitar a criação de gráficos interativos diretamente a partir do R, sem a necessidade de programar em JavaScript. Ele é muito popular em aplicações de ciência de dados e dashboards.

O plotly é construído sobre a tecnologia D3.js (Data-Driven Documents) e o WebGL, que permitem a renderização de gráficos diretamente em navegadores com alto desempenho e interação. Ele é altamente compatível com outras ferramentas de visualização no R, como o ggplot2, o que permite transformar gráficos estáticos em gráficos interativos com facilidade. É altamente versátil e pode ser utilizado tanto para visualizações simples como para gráficos complexos e tridimensionais, tudo com um alto nível de interatividade e personalização.

Principais Recursos do plotly

  1. Interatividade: Gráficos podem ser manipulados pelos usuários, incluindo zoom, rotação (para gráficos 3D), seleção de dados, e adição de anotações.

  2. Suporte a gráficos 3D: Permite a criação de gráficos tridimensionais como superfícies, dispersões e malhas (meshes) que podem ser rotacionados e explorados.

  3. Facilidade de uso: Simples de integrar com gráficos criados pelo ggplot2 e também com gráficos nativos do R, como os criados pela função plot_ly.

  4. Gráficos interativos e responsivos: Os gráficos se ajustam automaticamente ao tamanho da tela, o que os torna ideais para aplicações WEB e dashboards.

  5. Exportação: Permite exportar gráficos em formatos estáticos como PNG, SVG, PDF, além de integrá-los diretamente em documentos HTML interativos.

  6. Suporte a vários tipos de gráficos:

    • Gráficos de dispersão (scatter plots)
    • Gráficos de linhas
    • Gráficos de barras
    • Histogramas
    • Superfícies 3D
    • Gráficos de calor (heatmaps)
    • Gráficos de violino e boxplots
    • Gráficos de radar, etc.

Visite o site: https://plot.ly/r/ para consultar o ‘Guia de Referência’ e uma variedade de exemplos ilustrando os recursos do pacote.

O livro: ‘Interactive web-based data visualization with R, plotly, and shiny’ está disponível gratuitamente em https://plotly-r.com/ . A obra é dedicada à criação de visualizações interativas baseadas na WEB utilizando o R, focando principalmente no pacote plotly e na integração com o Shiny.

A toolbar (barra de ferramentas) nos gráficos criados com plotly oferece uma série de funcionalidades interativas que melhoram a experiência do usuário ao explorar os dados de maneira dinâmica. Esses recursos tornam a visualização não apenas mais atraente, mas também mais funcional, facilitando a análise de dados complexos de forma acessível. O usuário também pode personalizar a toolbar escolhendo quais ferramentas e recursos devem ser exibidos, utilizando o parâmetro config() na função plot_ly() (por exemplo, esconder a toolbar ou desativar certas funções).

Aqui estão os principais recursos e botões:

  1. Zoom:
    • O botão de zoom permite que o usuário selecione uma área do gráfico para ampliar. Após clicar no botão, você pode clicar e arrastar para criar um retângulo em torno da área que deseja ampliar.
  2. Pan:
    • O botão de pan permite que o usuário mova o gráfico. Isso é útil após usar o zoom, permitindo que você explore diferentes partes do gráfico sem ter que redefinir o zoom.
  3. Reset Axes:
    • Este botão redefine o gráfico para suas escalas e limites originais, permitindo que você retorne rapidamente à visualização padrão.
  4. Download:
    • O botão de download permite exportar o gráfico em diferentes formatos, como PNG, JPEG, PDF ou SVG. Essa funcionalidade é especialmente útil para incluir gráficos em relatórios ou apresentações.
  5. Select:
    • O recurso de seleção permite que o usuário selecione pontos ou uma área específica do gráfico. Dependendo do tipo de gráfico, essa seleção pode ser usada para destacar dados ou filtrar informações.
  6. Hover Information:
    • Ao passar o mouse sobre os pontos do gráfico, informações adicionais sobre os dados são exibidas. Isso pode incluir valores numéricos, rótulos e outras informações contextuais que ajudam na interpretação dos dados.
  7. Legend (Legenda):
    • A legenda permite ao usuário ativar ou desativar a visibilidade de diferentes séries de dados no gráfico, facilitando a comparação entre categorias ou grupos.
