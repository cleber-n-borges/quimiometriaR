Caderno de Quimiometria com R

ℚℝ 1 Comece a Usar o R

Começar a usar o R pode ser uma experiência empolgante e enriquecedora, especialmente se você está interessado em análise de dados e estatística. Aqui está um guia para você iniciar com o R:

1.1 Instale o R e RStudio

Siga as instruções:

  • Instale o R:

    1. Acesse o site do CRAN (Comprehensive R Archive Network)4.
    2. Selecione o link para o seu sistema operacional (Windows, macOS, Linux).
    3. Baixe e execute o instalador, seguindo as instruções na tela.

  • Instale o RStudio (opcional, mas fortemente recomendado):

    1. Vá para o site do RStudio5.
    2. Baixe a versão gratuita (RStudio Desktop Open Source License).
    3. Instale o RStudio após a instalação do R. RStudio fornece um ambiente de desenvolvimento integrado (IDE) que facilita o uso do R.

  • Abra o RStudio:

    1. Ao abrir o RStudio, você verá várias janelas, incluindo o console, o editor de scripts, o ambiente de trabalho e o painel de arquivos.
    2. Você já poderá executar comandos no console.

Dica: Não se esqueça de visitar o site oficial do R: https://www.r-project.org/.

1.2 Conheça o Básico do R

Quando você usa uma função nativa do R, você segue uma sintaxe que envolve chamar a função pelo seu nome e passar argumentos entre parênteses.

Sintaxe geral:

nome_da_funcao(argumento1, argumento2, ...)

  • Parênteses: Envolvem os argumentos da função.
  • Argumentos: Valores ou variáveis que a função precisa para executar sua tarefa.

Veja um exemplo:

  • Executar Códigos Simples: Experimente comandos básicos no console do RStudio:

    # Criar um vetor
x <- c(1, 2, 3, 4, 5)

# Calcular a média
mean(x)

# Calcular o desvio padrão
sd(x)

Aqui está a descrição dessa estrutura básica:

  • #: o sinal (chamado cerquilha) indica que a linha contém um comentário explicativo
  • x: é um nome que o usuário escolhe para receber o resultado
  • <-: é o operação de atribuição (que é um caso muito particular de função!)
  • c(): é a função que combina valores em um vetor
  • 1, 2, 3, 4, 5 são os dados digitados pelo usuário
  • mean(): é O nome da função que calcula o valor médio.
  • sd(): é O nome da função que calcula o valor do desvio padrão.

 

Copie o código acima e cole-o no console do R. Você obterá a seguinte saída como resultado:

> # Criar um vetor
> x <- c(1, 2, 3, 4, 5)
> 
> # Calcular a média
> mean(x)
[1] 3
> 
> # Calcular o desvio padrão
> sd(x)
[1] 1.581139
>
  • O valor da média dos dados é igual a 3 e o valor do desvio padrão é igual a 1,58.

 

Pronto! Você acabou de realizar sua primeira análise no R. Parabéns!

Continue assim e explore ainda mais as possibilidades que o R oferece.

1.4 A Conveniência do help()

O comando help() no R é uma ferramenta fundamental para obter informações detalhadas sobre funções, pacotes, e outros objetos do R. Utilizar help() pode melhorar significativamente a eficiência e a eficácia na utilização do R, pois fornece detalhes cruciais sobre como usar as funções e interpretar seus resultados.

Dica: Caso você queira os ler os manuais em português, pergunte ao ChatGPT:

“Como ajustar o R para ter os manuais em pt_BR?”

O ChatGPT explicará muitos detalhes sobre o comando abaixo:

Sys.setenv(LANG = "pt_BR.UTF-8")

Como Usar o Comando help():

  1. Obter Ajuda para uma Função Específica

Para obter informações sobre uma função específica, use help() seguido do nome da função. Por exemplo, para obter ajuda sobre a função mean(), você pode usar:

help(mean)

Ou, você também pode usar o símbolo de interrogação como uma alternativa:

?mean
  1. Obter Ajuda para um Pacote

Se você deseja obter uma lista de funções e informações sobre um pacote específico, você pode usar:

help(package = "cluster")

Isso exibe uma lista de todas as funções e datasets disponíveis no pacote cluster, além de um link para a documentação do pacote.

  1. Pesquisar por Funções e Conceitos

Para buscar uma função ou conceito sem saber o nome exato, você pode usar help.search(). Por exemplo, para procurar funções relacionadas a “linear regression”, você pode usar:

help.search("linear regression")

Alternativamente, o comando ?? faz a mesma busca e pode ser usado como:

?? linear regression
  1. Acessar a Documentação do Pacote

Muitos pacotes têm um arquivo de documentação que pode ser acessado diretamente com:

browseVignettes(package = "ggplot2")

Isso abre a página de vignettes do pacote no navegador, fornecendo documentação detalhada e exemplos.

  1. Metainformação

E evidentemente, execute o comando no R: help( help ) para mais detalhes.

Conveniência da Prática

  1. Acesso Imediato à Documentação: Utilizando help(), você pode acessar rapidamente a documentação das funções e pacotes sem sair do ambiente de programação. Isso economiza tempo e melhora a eficiência na escrita de código.

  2. Compreensão dos Argumentos e Uso: A documentação fornece detalhes sobre os argumentos que uma função aceita, seu propósito, e exemplos de uso. Isso ajuda a evitar erros e a entender como usar as funções corretamente.

  3. Exploração de Pacotes: Quando você instala novos pacotes, help() permite explorar as funções disponíveis e entender o que cada pacote oferece, facilitando a integração de novos recursos em seu trabalho.

  4. Resolução de Problemas: Se você encontrar problemas ao usar uma função, a documentação geralmente inclui informações sobre erros comuns e como resolvê-los, o que pode ajudar a depurar o código.

  5. Aprendizado Contínuo: Consultar a documentação regularmente ajuda a aprender mais sobre as funcionalidades do R e a manter-se atualizado sobre novos pacotes e métodos.

  6. Documentação Completa: O uso de help() proporciona acesso a documentação completa, incluindo descrições, argumentos, detalhes e exemplos, que pode ser mais abrangente do que uma simples busca na web.

Exemplo de Uso

Vamos dizer que você está trabalhando com a função lm() para regressão linear e deseja entender mais sobre seus argumentos e exemplos. Você pode digitar:

?lm

Isso abrirá a página de ajuda da função lm(), onde você encontrará uma descrição completa da função, os parâmetros que ela aceita, exemplos de uso e detalhes sobre a saída.

Dica: Pergunte ao ChatGPT como se estrutura uma página de ajuda sobre função R.

 

Em resumo, o comando help() no R é uma ferramenta essencial para a exploração e utilização eficiente das funcionalidades do R, oferecendo acesso direto à documentação e facilitando o desenvolvimento e análise no ambiente R.

1.5 Manuseie Arquivos

Saber manusear arquivos em R é fundamental para qualquer análise de dados, pois permite a importação e exportação de dados de e para diversos formatos.

  • Importação com read.table(): Essa função é essencial para carregar dados de arquivos de texto (como CSV ou TSV) para dentro do R. Sem a capacidade de importar dados, seria impossível realizar análises com informações que estão armazenadas em arquivos externos.

  • Exportação com write.table(): Depois de realizar análises ou manipulações, a função write.table() permite salvar os resultados em arquivos de texto. Isso é importante para compartilhar resultados, criar relatórios ou armazenar dados para uso futuro.

Essas funções facilitam a integração de R com outros sistemas e garantem que dados possam ser facilmente movidos e utilizados entre diferentes plataformas.

A função recebe como argumento o caminho do arquivo, o qual deve conter os dados como texto puro e em formato tabular e retorna um data.frame a partir dele, com casos correspondentes a linhas e variáveis a campos da tabela.

  1. read.table()

  • Descrição: A função read.table() é utilizada para importar dados de arquivos de texto contendo dados organizados em formato tabular delimitados por tabulação, espaços, ou outros delimitadores. Ela recebe como argumento o caminho do arquivo e retorna um objeto do tipo data.frame, onde cada linha do arquivo corresponde a uma observação e cada coluna a uma variável. A função é flexível e permite ajustar os parâmetros delimitadores de campo, nomes de colunas e tipos de dados para adequar-se à estrutura do arquivo de entrada.

  • Sintaxe:

read.table(file, header = FALSE, sep = "", ...)

  • file: O nome do arquivo ou URL.

  • header: Indica se o primeiro registro contém os nomes das colunas (TRUE ou FALSE).

  • sep: O delimitador dos campos, como “,” para CSV, “\t” para tabulação, ou ” ” para espaço.

  • …: Outros argumentos como colClasses, nrows, skip, etc.

  • Exemplo:

data <- read.table("dados.txt", header = TRUE, sep = "\t")
  1. write.table()

  • Descrição: A função write.table() é utilizada para exportar um objeto data.frame para um arquivo de texto em formato tabular. É frequentemente usada para salvar resultados de análises ou manipulações de dados. Ela recebe como argumento o caminho do arquivo de destino e o data.frame a ser salvo, e permite ajustar parâmetros como o delimitador de campo, a inclusão de cabeçalhos de coluna e amostras e o formato dos dados. O resultado é um arquivo de texto onde cada linha representa uma observação e cada coluna representa uma variável, estruturado de acordo com as especificações fornecidas.

  • Sintaxe:

write.table(x, file, sep = " ", row.names = TRUE, col.names = TRUE, ...)

  • x: O objeto a ser exportado, geralmente um data frame ou matriz.

  • file: O nome do arquivo para o qual os dados serão exportados.

  • sep: O delimitador a ser usado entre os campos no arquivo de saída.

  • row.names: Indica se os nomes das linhas devem ser incluídos.

  • col.names: Indica se os nomes das colunas devem ser incluídos.

  • Exemplo:

write.table(data, "dados_exportados.txt", sep = ",", row.names = FALSE)

Dica: Essas funções possuem variações como read.csv(), write.csv() e scan() para formatos específicos e leitura mais controlada.

A leitura com atenção do manual sobre essas funções é obrigatória para qualquer analista!

1.6 Salve e Restaure seu Workspace

Para garantir que todo o seu trabalho no R seja preservado e que você possa retomar suas atividades sem perder dados ou configurações, é importante salvar seu ambiente de trabalho - Workspace - regularmente. Aqui estão os passos para salvar seus dados, o histórico de comandos e a sessão de trabalho no R:

  1. Salve sua Sessão de Trabalho (Workspace): Para salvar tanto o workspace quanto o histórico de comandos, isto é, o ambiente completo, você pode usar a função save.image, que cria um arquivo .RData com todos os objetos e dados atuais:
save.image(file = "sessao_trabalho.RData")
  1. Salvar Objetos em um Arquivo RData: Para salvar todos os objetos do seu workspace (variáveis, funções, etc.) de uma vez, utilize a função save:
save(list = ls(), file = "meu_workspace.RData")

Você também pode usar save para salvar objetos individualmente se necessário.

  1. Salve o Histórico de Comandos: O R mantém um histórico dos comandos que você executou durante sua sessão. Para salvar este histórico, você pode usar a função savehistory:
savehistory(file = "meu_historico.Rhistory")

Isso criará um arquivo .Rhistory contendo todos os comandos que você digitou.

  1. Restaurar a Sessão: Quando você quiser retomar seu trabalho, carregue o arquivo .RData com a função load:
load("sessao_trabalho.RData")

Certifique-se de salvar seu trabalho frequentemente para evitar perdas e garantir que você possa continuar de onde parou. Isso é especialmente importante quando você está lidando com análises complexas e dados valiosos.

1.7 Instale Pacotes R

  • Instalar Pacotes:

Pacotes são conjuntos de ferramentas adicionais que você pode instalar no R para expandir suas funcionalidades. Eles são essenciais para tornar o R uma ferramenta poderosa, pois permitem que você adicione funções e procedimentos específicos para diferentes áreas da ciência e da análise de dados. Com pacotes, você pode personalizar o R para atender às suas necessidades específicas, tornando-o ainda mais eficiente e versátil.”

No console do RStudio, você pode instalar pacotes com o comando install.packages(). Por exemplo:

# Instala o pacote mdatools para preprocessamento,
# explorar e analisar dados multivariados. 
install.packages("mdatools") 
  
# Instala o pacote ggplot2 para visualização de dados
install.packages("ggplot2")
  • Carregar Pacotes:

Após instalar um pacote no R, ele não estará disponível automaticamente para uso. Para começar a usar as funções e recursos do pacote, você precisa carregá-lo na sessão atual com o comando library(). Por exemplo, para carregar o pacote chamado ‘cluster’, você deve usar o comando library(cluster). Lembre-se de fazer isso sempre que iniciar uma nova sessão do R e quiser usar as funcionalidades do pacote.”

# Carrega o pacote "cluster"
library(cluster)

# Carrega o pacote "class"
library(class)
